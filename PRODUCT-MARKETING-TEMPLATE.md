## 2. Main Features & Functionality

### Core Features (Primary Features Users Will Use Daily)

#### Feature 1: Live Countdown Timer
- **Description**: Every holiday and event displays a beautiful live countdown showing days, hours, minutes, and seconds until the moment arrives. The "Next Holiday" card features a prominent real-time timer.
- **How It Works**: 
  1. User adds holidays/events to their list
  2. App calculates time remaining in real-time
  3. The next upcoming holiday is featured prominently with a large countdown display
  4. Days, hours, minutes, seconds all update live
  5. For recurring events (birthdays, anniversaries), app automatically calculates the next occurrence
- **User Benefit**: Creates excitement and anticipation for upcoming events. Users can watch the countdown tick down in real-time, building excitement as the day approaches.
- **Screenshot/Wireframe Location**: [________________]

#### Feature 2: Smart Holiday Grouping
- **Description**: Holidays are intelligently grouped by time period: Next 3 Months, Next 6 Months, Next 12 Months, and Later. This helps users focus on what's coming soon while keeping track of the full year.
- **How It Works**:
  1. App automatically calculates which time period each holiday falls into
  2. Holidays are sorted chronologically within each group
  3. The very next holiday is displayed prominently as a "featured" card
  4. Past one-time events are automatically hidden; recurring events show next occurrence
- **User Benefit**: Organized view that prioritizes what's coming soon without losing sight of future events. No clutter from past events.
- **Screenshot/Wireframe Location**: [________________]

#### Feature 3: One-Tap Country Holiday Import
- **Description**: Users can browse and import all public holidays for their country with a single tap. The app detects your location and suggests your country first.
- **How It Works**:
  1. User taps "Browse" from the holiday list
  2. App detects user's country and shows it as a suggestion
  3. User can search/browse 100+ countries
  4. Each country shows the number of holidays available
  5. One tap imports all public holidays with names, dates, and descriptions
  6. Holidays are sourced from official government data
- **User Benefit**: Get started instantly without manually entering holidays. All your country's public holidays added in seconds with accurate dates and descriptions.
- **Screenshot/Wireframe Location**: [________________]

#### Feature 4: Custom Event Creation
- **Description**: Users can create custom holidays and celebrations with a beautiful date picker, emoji icons, recurrence options, and notification reminders.
- **How It Works**:
  1. User taps the + FAB button
  2. Selects an emoji icon for the event
  3. Enters the event name
  4. Uses the iOS-style wheel picker to select date
  5. Sets recurrence (Never, Yearly, Monthly, Weekly)
  6. Configures reminder notifications (On day, 1 day before, 3 days, 1 week, 2 weeks)
  7. Adds optional notes/description
  8. Saves the event
- **User Benefit**: Complete customization for any personal milestone - birthdays, anniversaries, vacations, work deadlines. Set it once and the app handles recurring events automatically.
- **Screenshot/Wireframe Location**: [________________]

#### Feature 5: Smart Leave Planner (AI-Powered Optimization)
- **Description**: Revolutionary feature that analyzes your public holidays and suggests the most efficient way to use your annual leave. Shows how to turn 1 leave day into 4-day weekends.
- **How It Works**:
  1. User sets their annual leave budget (e.g., 14 days)
  2. Algorithm analyzes all holidays against the calendar
  3. Identifies "long weekend opportunities" based on holiday placement:
     - Monday holidays = natural 3-day weekend
     - Tuesday holidays = take Monday off for 4-day weekend
     - Wednesday holidays = take surrounding days for mega break
     - Thursday holidays = take Friday off for 4-day weekend
     - Friday holidays = natural 3-day weekend
  4. Shows efficiency score (days off per leave day used)
  5. User selects opportunities to plan
  6. Can extend/shrink date ranges interactively
  7. Tracks remaining leave days budget
  8. Integrates with AI for destination recommendations
- **User Benefit**: Maximize your time off intelligently. Never waste a leave day - always get the most vacation for your annual leave. Visual planning makes it easy to see exactly when to take time off.
- **Screenshot/Wireframe Location**: [________________]

#### Feature 6: Calendar Import (ICS & CSV)
- **Description**: Import existing calendars and events from any calendar app or spreadsheet. Supports standard ICS format and CSV files.
- **How It Works**:
  1. User navigates to Import screen
  2. Selects file type (ICS or CSV)
  3. Uploads file from device
  4. App parses events with names, dates, descriptions
  5. Shows import results (success count, errors, warnings)
  6. Events appear in holiday list
- **User Benefit**: No need to manually re-enter events. Export from Apple Calendar, Google Calendar, Outlook, or any calendar app and import instantly.
- **Screenshot/Wireframe Location**: [________________]

#### Feature 7: Home Screen Widgets (Customizable)
- **Description**: Beautiful home screen widgets that show your countdown at a glance. Multiple sizes (Small, Medium, Large) with customizable themes, effects, and colors.
- **How It Works**:
  1. User navigates to Widget Configuration screen
  2. Selects widget size
  3. Chooses theme (Winter Wonderland, Neon Party, Minimalist, etc.)
  4. Adds celebration effects (Snow, Confetti, Sparkle)
  5. Customizes text color
  6. Toggles date and unit display
  7. Adds widget to home screen
- **User Benefit**: See your countdown without opening the app. Beautiful widgets that make your home screen more exciting. Customize to match your style.
- **Screenshot/Wireframe Location**: [________________]

#### Feature 8: Telegram Mini App Integration
- **Description**: App works natively as a Telegram Mini App with push notifications for upcoming events.

## 3. Special & Unique Features

### Features That Differentiate Us from Competitors

#### Unique Feature 1: Smart Leave Planner with Visual Calendar
- **Description**: No other countdown app helps you maximize your annual leave. Our algorithm identifies the most efficient days to take off based on public holiday placement.

- **Technical Details**: Algorithm analyzes day-of-week for each holiday, calculates adjacent weekends, scores efficiency (days off / leave days used), supports interactive date range manipulation with localStorage persistence.
- **User Benefit**: Users get more vacation time without using more leave days. A Thursday holiday + 1 Friday leave = 4-day weekend. Our app makes this obvious and easy to plan.
