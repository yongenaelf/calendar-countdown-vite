2. Main Features & Functionality

Core Features (Primary Features Users Will Use Daily)

Feature 1: Plant Your Manifestation (AI-Powered)
- Description: Users type in their manifestation/intention, and AI transforms it into a beautifully worded manifestation statement. The user then "plants" it, and AI generates a unique beautiful plant seed with beautiful 3D claymorphism animation.
- How It Works: 
  1. User opens app and taps "Plant New Manifestation"
  2. User types their intention (e.g., "I want to find my dream job")
  3. AI transforms the input into a beautifully worded manifestation statement
  4. User reviews and confirms the manifestation
  5. AI generates a unique plant seed with beautiful 3D claymorphism animation
  6. Plant appears in user's garden as a seed
- User Benefit: Makes setting intentions feel beautiful and tangible. The visual representation (plant) creates emotional connection to the manifestation.

Feature 2: Daily Watering with AI-Generated Affirmations
- Description: Users "water" their plants daily by holding and reading an AI-personalized affirmation. AI generates affirmations specifically tailored to each manifestation and user context. This is the core daily ritual that helps plants grow.
- How It Works:
  1. User opens their garden and sees their plants
  2. User selects a plant to water

4. Design System & Aesthetic Details

Brand Identity: "Mystical Minimalism"

Note: "Mystical Minimalism" is the internal design system name. For App Store and public-facing materials, use "beautiful minimalism" or "organic minimalism" to ensure compliance.

Core Aesthetic:
- Clean modern UI meets whimsical, organic elements
- Feelings: Growth, peace, beauty
- Balance: Professional but beautiful, minimal but warm
  
Typography System

Primary Font: Nunito (Google Fonts)

Header Style:
- Weight: font-extrabold (800) or font-black (900)
- Tracking: tracking-tight
- Use case: Main headings, plant names, manifestation titles
  
Body Style:
- Weight: font-medium (500) or font-semibold (600)
- Use case: Affirmation text, descriptions, UI copy
- Rationale: Ensures readability while maintaining warmth
  
Labels/Badges:
- Weight: font-black
- Transform: uppercase
- Tracking: tracking-widest
- Size: text-[10px]
- Use case: Growth stage labels, badges, metadata
- Rationale: Creates "botanical archive" aesthetic

Design Principles & UI Elements

Soft Geometry:
- Extremely rounded corners: rounded-2xl, rounded-3xl, or rounded-full
- Rationale: Emphasizes organic, natural shapes (like plants)
- Use case: Cards, buttons, plant containers, modals
  
The "Cream Card":
- Background: #FDFBF7 (Cream-white)
- Purpose: Container for 3D plant renders
- Rationale: Makes plant colors and shadows "pop" against neutral background
- Use case: Plant detail views, featured plant cards
  
Glassmorphism:
- Effect: backdrop-blur-xl with semi-transparent backgrounds
- Use case: Overlays, navigation bars, modals, floating elements
- Rationale: Creates depth and modern, ethereal feel
  
Colored Shadows:
- Style: Soft, colored shadows instead of harsh black
- Examples: shadow-emerald-200/50, shadow-rose-300/50
- Rationale: Maintains "glowing aura" around elements
- Use case: Plant cards, buttons, floating elements
  
Motion & Micro-interactions

Floating Animation (animate-float):
- Effect: Elements gently hover up and down
- Rationale: Creates "weightless" and delightful feel
- Use case: Plant renders, decorative elements
  
Breathe Animation (animate-breathe):
- Effect: Subtle scaling and brightness pulsing
- Rationale: Makes elements feel "alive"
- Use case: Plant seeds, waiting states, focus elements
  
Watering Interaction:
- Button: "Hold to Manifest" with filling progress bar
- Animation: "Wet shake" on plant when watering completes
- Rationale: Creates satisfying feedback and sense of completion
- Duration: 2-3 minutes recommended for meaningful affirmation practice
  
Fade-In Transitions:
- Easing: Custom cubic-bezier(0.34, 1.56, 0.64, 1)
- Effect: Slight overshoot for "springy" feel
- Rationale: Makes UI feel alive and organic, not mechanical
- Use case: All UI transitions, page loads, element appearances
  
Imagery Style

3D Claymorphism:
- Style: Soft 3D clay renders with studio lighting
- Rationale: Creates tactile, beautiful feel while maintaining modern aesthetic
- Technical: Generated renders with soft shadows, rounded forms, matte finish
  
Isolation:
- Effect: All plants isolated with no background
- Method: Client-side flood-fill background removal
- Result: Plants appear as floating specimens
- Rationale: Clean, focused presentation, easy to composite
  
Bioluminescence:
- Effect: Mature plants feature "glow" or "beautiful aura"
- Implementation: Described in AI generation prompts
- Rationale: Emphasizes beautiful nature of plants, creates visual interest
- Use case: Fully Bloomed stage plants, special/rare plants

6. Pricing Model

Pricing Structure

Free Tier:
- What's included?
  - 3 free plant slots per week
  - 3 watering sessions per plant per day
  - Same plant variations
  - Same animations
  - Access to core features (planting, watering, growth stages)
- Limitations:
  - Limited to 3 plant slots per week (can plant 3 new manifestations per week)
  - Limited to 3 watering sessions per plant per day (can water each plant 3 times daily)
  - Limited total plants in encyclopedia (10 max saved plants)
  - No premium plant variations
  - No exclusive bioluminescent plants
  - Basic animations only
  - No cloud sync
- User Goal: Experience core value, understand product, convert to paid. Free tier allows users to experience the full planting and watering ritual, see plant growth, and understand the value proposition before upgrading.

Premium/Paid Tier:
- Price Point: 
  - $4.99/month (monthly subscription)
  - $39.99/year (annual subscription - saves ~33% vs. monthly)
  - Competitive with affirmation apps (ThinkUp, I Am, etc.)
